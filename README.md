# API de Gerenciamento de Usuários

Projeto de estudo para praticar desenvolvimento back-end com Node.js, Express e APIs REST.

## Objetivo

Criar uma API simples para cadastro, listagem, atualização e remoção de usuários, aplicando conceitos fundamentais de back-end.

## Tecnologias

- Node.js
- Express.js
- JavaScript
- JSON

## Funcionalidades planejadas

- Listar usuários
- Buscar usuário por identificador
- Criar usuário
- Atualizar usuário
- Remover usuário
- Validar campos obrigatórios
